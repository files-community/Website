Today we are releasing **Files Preview v4.0.36** for users of the preview version.

An update icon should be displayed in the top right corner of the app for existing users, while new users can get started from our [download](/download/) page. Additionally, you can help support the project by purchasing Files on the [Microsoft Store](ms-windows-store://pdp/?ProductId=9NSQD9PKV3SS&cid=FilesWebsite) or by sponsoring us on [GitHub](https://github.com/sponsors/yair100). Your support is greatly appreciated but entirely optional.

## What's New in Files v4.0.36

### Right Click Context Menu

You can now access quick actions by right clicking on the Home page, including options like toggling widgets and adding new panes. This dedicated menu makes it easier to customize your Home layout without leaving the page.

### Compression Attributes

Added support for applying compression attributes across multiple selected files from the Properties window.

### Details Pane

The Details Pane now includes a storage overview for Cloud Drives.

### Sidebar Drag and Drop

Added support for dragging sidebar items into other apps.

### Sidebar Toggle Action

Added a dedicated action to toggle the sidebar, giving you a faster way to show or hide navigation space when you want to focus on content. You can also access this action through the **Ctrl + B** keyboard shortcut.

### 7z Options

Added 7z dictionary and word size options in compression workflows. These advanced settings provide more control when tuning archive size versus compression speed.

### Scrolling

Added middle click to scroll, making navigation smoother in long directories and providing another familiar input method.

### Tabs

Added path hints for tabs with the same name, so it is easier to distinguish similarly named folders opened from different locations.

## Fixes

- Fixed an issue where thumbnails didn't load for MTP devices.
- Fixed an issue where the Third Party Libraries section was cut off.
- Fixed Status Center display when canceling extraction.
- Fixed an issue with opening Control Panel from desktop.
- Fixed the size of the Settings dialog close button.
- Fixed an issue where the Run With PowerShell action wasn't disabled in Recycle Bin.
